More about this project :)

This website has been created as a project during the week of the Summer Tech Experience at the Holberton School in Puerto Rico (2022). During this experience we learn how to create a website using code such as HTML and CSS. We used codepen.io to create our websites.

For my website, I decided to make it about the Capybara. Why? Why not! We could do anything for our websites so why not make it about Capybaras. I had fun making this project and I sure am going to make more projects like these in the future.
